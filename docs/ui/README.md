# 控制台与多端体验文档中心

## 目标
- 定义 Web 控制台、桌面客户端、浏览器扩展的 IA、交互流程与组件规范。
- 形成视觉与体验基线，支撑阶段1-4 的界面开发。

## 目录规划
1. `console_ia.md` —— 对话面板、任务时间线、截图流布局。
2. `components/` —— 通用组件（对话卡片、指标卡、日志视图）。
3. `desktop_client.md` —— 桌面端架构、离线缓存策略。
4. `extensions.md` —— 浏览器扩展交互与安全约束。
5. `collaboration.md` —— 多用户协作界面、权限 UI。
6. `browser_use_patterns.md` —— 参考 Browser Use 的 UI 模式（视觉高亮、行动建议、协作流程）。

- 阶段2 添加模板管理、偏好设置 UI 草案。
- 阶段3-4 扩展插件市场、监控告警、多端体验文档。
